## 🛠️ Session 1: Secure Foundations

In this initial phase, we established the **Golden Rules** for AI-assisted development. We do not delegate logic; we delegate the initial boilerplate under strict human supervision.

### Implemented Micro-services:

  - **QR Generator**: Buffer management and payload size limits (Security focus).
  - **Currency Converter**: Floating-point precision and type validation (Fintech focus).

### Technical Stack (Session 01)

  - **Runtime**: Node.js v18+
  - **Testing**: Mocha & Chai (Unit testing pyramid).
  - **AI**: Gemini 1.5 Flash (Logic modeling and edge-case detection).

-----

## 🛡️ AI Development Manifesto (RULES.md)

All code within this repository adheres to these non-negotiable guidelines:

1.  **Line-by-Line Review**: No AI-generated function reaches `main` without manual auditing.
2.  **Source Attribution**: Mandatory comment `// Generated by [AI] - reviewed by [Name]`.
3.  **Zero-Trust Security**: Input validation and explicit error handling in every handler.
4.  **Test-Driven Assisted Development**: Every AI-generated piece must be accompanied by a test suite to prevent regressions.
